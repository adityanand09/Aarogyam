# Aarogyam
Healthcare app

Shows the current medical condition based on the input provided using ApiMedic Symptom Checker API.

Home page options :

  Start diagnosing yourself or others.
  
  Add new past or current medical information with medicine pictures.
  
  Navigation panel :
    
    Login | Signup
    
    Profile
    
    History
    
    Signout
    
    
History options :
  
  Locally added history
  
  Searched through the application
